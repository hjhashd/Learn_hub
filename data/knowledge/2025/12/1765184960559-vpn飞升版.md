## 🔮 破阵心法（核心原理）

### 壹·虚空隧道之术（VPN 本质）

```
你的肉身（电脑）──加密封印──> 中介道场（VPS 服务器）──解封──> 外域（Google）
                  ↑                         ↓
              天道看不穿              以服务器身份行走
```

**关键妙处**：
- **加密封印**：你与服务器之间的传输用强加密术（VLESS-Reality），天道只能看到乱码。
- **身份伪装**：外域网站看到的是服务器 IP，而非你的真身。
- **隧道强制**：所有流量（游戏/浏览器/后台更新）都被强制塞入这条地下通道。

### 贰·Reality 伪装大法（防封核心）

裸奔的 WireGuard/OpenVPN 就像在天道眼皮底下舞剑——找死！  
Reality 协议妙在**"借壳上市"**：

```
你访问 YouTube → 装成访问 www.microsoft.com → 天道一看：
"哦，这娃在访问微软官网下载更新，放行！"
```

实则：流量早已在虚空中偷梁换柱，直达外域！

## 👹 禁术·施法步骤

### 第一式：筑基（连接服务器）

**推荐法器**：VS Code + Remote SSH 插件（比 FinalShell 这破烂强百倍）

```bash
# 在 VS Code 终端施展：
ssh ubuntu@你的服务器IP

# 切换到大佬模式（Root）
sudo -i

# 开启 BBR 加速阵法（让网速起飞）
echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf
echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf
sysctl -p
```

### 第二式：布阵（部署 X-UI 面板）

**法阵配置文件**（`docker-compose.yml`）：

```yaml
# filepath: /root/x-ui/docker-compose.yml
# 😈 此乃虚空传送门之核心阵眼，轻易勿动！

version: '3'
services:
  x-ui:
    image: ghcr.io/mhsanaei/3x-ui:latest
    container_name: x-ui
    volumes:
      - ./db/:/etc/x-ui/          # 法阵数据存于此
      - ./cert/:/root/cert/       # 伪装证书藏于此
    tmpfs:
      - /tmp
    restart: always
    network_mode: host            # 🔥 直接使用主机网络，性能拉满
    environment:
      XRAY_VMESS_AEAD_FORCED: "false"
```

**开坛做法**：

```bash
# 创建阵眼目录
mkdir -p /root/x-ui && cd /root/x-ui

# 将上述咒文写入 docker-compose.yml（在 VS Code 里直接新建文件）

# 运转大阵！
docker compose up -d

# 观察阵法是否正常运转
docker ps
```

**访问控制台**：  
打开浏览器输入 `http://你的IP:2053`，默认账密皆为 `admin`（记得立即改掉！）

### 第三式：结丹（配置 Reality 节点）

在 X-UI 面板 -> 入站列表 -> 添加入站：

```yaml
# 🔥 核心参数（切勿走火入魔）：

基本设置:
  协议: vless
  端口: 443              # 伪装成 HTTPS
  流控: xtls-rprx-vision  # 抗封神技

传输配置:
  安全性: reality
  Dest: www.microsoft.com:443  # 借微软的壳
  SNI: www.microsoft.com
  uTLS: chrome
  Private Key: [点击 Get New Key 生成]
  ShortId: [点击 0-f 生成]
```

**⚠️ 血的教训**：`Dest` 和 `SNI` 必须一致！否则会被踢出去！

### 第四式：开光（客户端连接）

**Windows**：下载 `v2rayN`，复制节点链接（`vless://...`），按 `Ctrl+V` 导入。  
**Android**：下载 `v2rayNG`（选 arm64 版本），扫描二维码。  
**iOS/iPad**：使用 `Shadowrocket`，扫码即可。

**激活法阵**：
- 右键托盘图标 → 系统代理 → 自动配置
- 访问 `ipinfo.io` 验证是否显示新加坡 IP

## 🗡️ 进阶秘术

### 壹·审计日志（监控设备行为）

为了知道"哪台设备在何时干了何事"，需分配不同身份标识：

1. 在 X-UI 面板编辑节点 → 客户端设置 → 添加多个 ID
2. 为每个 ID 设置不同 `email` 标签（如 `pc`/`ipad`/`phone`）
3. 在 Xray 设置 → 开启访问日志：
   ```
   访问日志路径: /etc/x-ui/access.log
   日志级别: info
   ```

**实时监控**：
```bash
# 实时查看法阵运转情况
tail -f /root/x-ui/db/access.log

# 查询某设备的行为
grep "email: ipad" /root/x-ui/db/access.log

# 查询谁在看 YouTube
grep "youtube" /root/x-ui/db/access.log
```

**日志格式解读**：
```
2025/11/27 15:30:00 accepted tcp:www.google.com:443 [vless] email: ipad
         ↑              ↑                          ↑
       时间          访问目标                   设备标识
```

### 贰·防走火入魔指南

| 禁忌行为 | 后果 | 解决之道 |
|---------|------|---------|
| 分享节点给百人 | IP 被封 | 自己独享最稳 |
| 发表敏感言论 | 实名追责 | 只看不说 |
