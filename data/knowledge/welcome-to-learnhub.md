# 欢迎使用 LearnHub

这是一个个人知识管理系统，帮助你整理和管理学习笔记。

## 介绍

LearnHub 是一个基于 Next.js 和 TypeScript 构建的现代知识管理应用。它支持：

- 📚 分类管理知识条目
- 🔍 全文搜索功能
- 📝 Markdown 格式支持
- 🌙 暗色模式
- 📱 响应式设计

## 功能特性

### 1. 智能分类
系统支持多级分类，让你的知识井井有条。

### 2. 快速搜索
通过关键词快速找到需要的知识点。

### 3. 优雅展示
支持 Markdown 渲染，代码高亮，让内容更清晰。

### 4. 目录导航
自动生成文章目录，方便快速跳转。

## 开始使用

点击左侧的笔记列表开始浏览，或者使用搜索框查找特定内容。

开始你的知识管理之旅吧！🚀
